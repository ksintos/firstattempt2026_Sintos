#### Installation

To replicate and run this project follow the following steps using Windows Powershell:

```bash
winget install OpenJS.NodeJS.LTS
nvm install lts
nvm use lts
https://github.com/ksintos/firstattempt2026_Sintos.git
cd alumni-portal
npm run dev
```
